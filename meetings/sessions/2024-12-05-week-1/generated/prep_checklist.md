#### Key Talking Points:

- Show before/after improvement for each framework
- Emphasize universal applicability across AI tools
- Maintain energy and enthusiasm

   ```
   - Some can't upload, others get corrupted images
   ```

2. **AI Tool Setup**:
   - Claude.ai account logged in
   - ChatGPT account logged in (backup)
   - Browser tabs pre-arranged
   - Copy-paste templates ready

3. **Demo Prompts Ready**:
   - Information extraction prompt (formatted and tested)
   - PRD generation prompt (CLEAR framework applied)
   - Test plan prompt (Role + Constraints applied)

#### Demo Flow Practice:
- [ ] Practice transitions between steps
- [ ] Test actual AI responses to prompts
- [ ] Prepare commentary for each AI output
- [ ] Have backup examples if demo fails

### Hands-On Practice Setup (Remy + Bora support)
#### Preparation Required:
1. **Example Bank**: Prepare 3-4 messy work examples for different roles:
   - Design: "Make the UI better"
   - Development: "Fix the bug"
   - Product: "Users don't like the feature"
   - Content: "Write something engaging"

2. **CLEAR Framework Template** (ready to share):
   ```
   Context: [What's the situation?]
   Length: [How much detail needed?]
   Examples: [What good outputs look like?]
   Action: [What specific task?]
   Role: [What expert perspective?]
   ```

3. **Chain of Thought Starter**:
   ```
   "Let's think through this step by step:
   1. First, [identify core issue]
   2. Then, [explore approaches]
   3. Finally, [evaluate and decide]"
   ```
