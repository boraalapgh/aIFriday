Prepare comprehensive materials for an upcoming AI Friday session using the orchestrated agent system with specialized coordination and logistics management.

This command triggers the **Meeting Preparation Workflow** using the AI Friday orchestrator and the meeting-coordinator sub-agent.

## What This Command Does

1. **Identifies Target Session**: Determines which session needs preparation materials
2. **Reviews Current State**: Analyzes existing content plan and speaker confirmations
3. **Invokes Meeting Coordinator**: Specialized sub-agent handles all logistics and preparation
4. **Generates Complete Prep Package**: All materials needed for successful session execution
5. **Coordinates Final Details**: Speaker communications, technical setup, and facilitation support

## Preparation Materials Created

### Host Preparation Package
- **Pre-session checklist**: Timeline-based setup tasks with technical requirements
- **Facilitation guide**: Discussion prompts, transition scripts, and timing management
- **Backup procedures**: Baseline topic preparation and emergency protocols
- **Technical troubleshooting**: Common issues and resolution steps

### Speaker Coordination Materials
- **Individual prep guides**: Customized requirements for each speaker
- **Technical verification**: Demo environment testing and setup confirmation
- **Communication timeline**: Preparation milestones and check-in schedule
- **Day-of logistics**: Final timing, platform access, and support contacts

### Communication Assets
- **Speaker confirmations**: Professional welcome and timeline setting
- **Participant announcements**: Session promotion and logistics sharing
- **Reminder templates**: Multiple touchpoints leading up to session
- **Follow-up materials**: Thank you messages and resource sharing protocols

### Technical Coordination
- **Platform configuration**: Meeting setup, recording, and accessibility features
- **Demo requirements**: Specific technical needs for each presentation
- **Quality assurance**: Testing procedures and backup solutions
- **Support resources**: Technical help contacts and troubleshooting guides

## Logistics Intelligence

### Session Analysis
The meeting-coordinator automatically:
- Reviews finalized content plan from content-curator
- Analyzes technical requirements for each presentation
- Identifies potential logistics challenges and solutions
- Coordinates timing based on content complexity and interaction needs

### Communication Timeline
**Automated scheduling for**:
- **2 weeks before**: Initial speaker confirmations and preparation timeline
- **1 week before**: Detailed prep materials and technical checks
- **Day before**: Final reminders and logistics confirmation
- **Day of**: Session access, agenda distribution, and real-time support

### Quality Assurance
- All speaker technical requirements verified and documented
- Platform configuration tested for optimal experience
- Backup plans prepared for common scenarios
- Communication materials personalized for speakers and audience
